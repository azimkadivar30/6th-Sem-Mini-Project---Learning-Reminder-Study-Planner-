# 📚 Learning Reminder & Study Planner

## 📌 Project Description
The Learning Reminder and Study Planner is a web-based application developed to help students manage their academic tasks efficiently. 

Students often face problems like poor time management, forgetting deadlines, and lack of proper study planning. This system provides a simple and effective solution by allowing users to organize subjects, create study tasks, and set reminders.

The application also includes advanced features such as a Study Session Timer to track study time, Progress Analytics to monitor performance, and a Reward System with points and badges to motivate students.

---

## 🎯 Objectives
- To help students manage study tasks and schedules  
- To provide subject-wise academic planning  
- To track study time using a timer  
- To improve productivity and consistency  
- To motivate students using rewards and streak tracking  

---

## 🚀 Key Features
- 📖 Subject Management (Add, Edit, Delete)  
- 📝 Study Planner (Task creation with deadlines)  
- ⏰ Reminder System  
- ⏱️ Study Session Timer  
- 📊 Progress Analytics  
- 🏆 Reward System (Points & Badges)  
- 🔥 Study Streak Tracker  
- 👤 Profile Management  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL / MariaDB  
- Server: XAMPP  

---

## 🗄️ Database Tables
- users  
- subjects  
- tasks  
- reminders  
- study_sessions  

---

## ⚙️ How to Run the Project
1. Install XAMPP  
2. Move project folder to `htdocs`  
3. Start Apache and MySQL  
4. Import database in phpMyAdmin  
5. Run in browser:  
   http://localhost/your-folder-name
